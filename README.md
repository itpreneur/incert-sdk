Incert SDK 
=========

Blockchain certificate API

## Installation

  `npm install incert-sdk`

## Usage

    var numFormatter = require('incert-sdk');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`


## Tests

  `npm test`
